程序中总共实现了4条flash通讯指令，最终实现情况如下：

1. 读取数据指令
将i_cmd_valid置高，同时将0x03赋值给i_cmd后，触发读取数据指令
共有1+3+size个字节的周期，包括1个字节的读数据指令（0x03），3个字节的地址，size个字节的连读数据
每次读取一个完整字节后，输出1个clk周期的o_data_valid正脉冲，在o_data_valid为高时，读取o_data_out上的数据
读指令执行完成后会输出1个clk周期的o_cmd_ack正脉冲

2. 页编程指令
将i_cmd_valid置高，同时将0x02赋值给i_cmd后，触发页编程指令
先发送1个字节的写使能指令（0x06）
再开始页编程操作，共有1+3+size个字节的周期，包括1个字节的页编程指令（0x02），3个字节的地址，size个字节的连续写入数据
每次传输完1个字节数据后，输出1个clk周期的o_data_req正脉冲，用于请求新的写入数据
最后开始读取状态寄存器，共有1+1个字节的周期，包括1个字节的读取状态寄存器指令（0x05），1个字节的状态寄存器数值内容
传输完状态寄存器数值后，输出1个clk周期的o_data_valid正脉冲，在o_data_valid为高时，读取o_data_out上的数据，如果读取内容为未擦除完成，则一直循环执行查询状态寄存器指令
页编程指令执行完成后会输出1个clk周期的o_cmd_ack正脉冲

3. 扇区擦除指令
将i_cmd_valid置高，同时将0xd8赋值给i_cmd后，触发扇区擦除指令
先发送1个字节的写使能指令（0x06）
再开始扇区擦除操作，共有1+3个字节的周期，包括1个字节的扇区擦除指令（0xd8），3个字节地址
最后开始读取状态寄存器，共有1+1个字节的周期，包括1个字节的读取状态寄存器指令（0x05），1个字节的状态寄存器数值内容
传输完状态寄存器数值后，输出1个clk周期的o_data_valid正脉冲，在o_data_valid为高时，读取o_data_out上的数据，如果读取内容为未擦除完成，则一直循环执行查询状态寄存器指令
读指令执行完成后会输出1个clk周期的o_cmd_ack正脉冲

4. 块擦除指令
将i_cmd_valid置高，同时将0xc7赋值给i_cmd后，触发扇区擦除指令
先发送1个字节的写使能指令（0x06）
再发送1个字节的块擦除指令（0xc7）
最后开始读取状态寄存器，共有1+1个字节的周期，包括1个字节的读取状态寄存器指令（0x05），1个字节的状态寄存器数值内容
传输完状态寄存器数值后，输出1个clk周期的o_data_valid正脉冲，在o_data_valid为高时，读取o_data_out上的数据，如果读取内容为未擦除完成，则一直循环执行查询状态寄存器指令
读指令执行完成后会输出1个clk周期的o_cmd_ack正脉冲
